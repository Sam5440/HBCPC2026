## 题目 I. 山海关老冰糕

冰凉清甜的老冰糕是山海关特色。老冰糕工厂里有 N 台生产线，第 i 台生产线生产一份老冰糕需要耗时 $t_{i}$ 分钟。所有生产线同时开工，且可以并行工作。

现在工厂接到了一个巨大的夏日订单，需要在最短的时间内生产出至少 M 份老冰糕。请问工厂最少需要多少分钟才能完成订单？

### 输入

第一行包含两个整数 $N$ 和 $M(1 \leq N \leq 10^5, 1 \leq M < 10^{12})$ ，分别表示生产线数量和订单需求量。  
第二行包含 N 个整数 $t_{1}, t_{2}, \ldots, t_{N} (1 \leq t_{i} \leq 10^{6})$ ，表示每条生产线生产一根冰棍的耗时。

### 输出

输出一个整数，表示完成订单所需的最少分钟数。

### 样例

| standard input | standard output |
| -------------- | --------------- |
| 3 10 2 3 5     | 10              |
